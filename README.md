# JEM DISCORD BOT

An AI Discord bot built with Python, Discord.py, and Google Gemini.

## Features

- AI conversations
- Custom personality (MY PERSONALITY o((>ω< ))o)
- Trigger using `hi jem`
- Powered by Gemini

## Tech Stack

- Python
- Discord.py
- Google Gemini API
- python-dotenv

## Setup

Clone the repository:

```bash
git clone https://github.com/jananiwashere/jem-discord-bot.git
```

Install dependencies:

```bash
py -m pip install -r requirements.txt
```

Create a `.env` file:

```env
DISCORD_TOKEN=YOUR_TOKEN
GEMINI_API_KEY=YOUR_KEY
```

Run:

```bash
py main.py
```
